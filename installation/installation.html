# Installation

VISOR is written in python 3.6 and uses bash to call sub-processes, thus requiring a working python 3.6 environment and a unix machine in order to be installed from source and run properly. VISOR is also available as a Docker container.

# Installation from source

A python 3.6 environment can be installed using Anaconda.

curl -LO http://repo.continuum.io/miniconda/Miniconda-latest-Linux-x86_64.sh
bash Miniconda-latest-Linux-x86_64.sh && rm Miniconda-latest-Linux-x86_64.sh
cd ~ && source .bashrc
conda create -y -n visorenv python=3.6
source activate visorenv

Once the python 3.6 environment has been created, conda can be used to install VISOR's dependencies.

conda install -n visorenv -c bioconda samtools wgsim pbsim bwa minimap2 pybedtools pysam pyfaidx

VISOR can then be cloned and installed.

git clone https://github.com/davidebolo1993/VISOR.git
cd VISOR
python setup.py install
VISOR --help

# Docker container

For Docker users, VISOR is also available as a Docker image. One can pull the image as below.

#will require sudo privileges, probably
docker pull davidebolo1993/visor
docker run davidebolo1993/visor VISOR --help
